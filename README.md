## 运行方式

1. 安装 Python 3.x。
2. 克隆仓库后进入项目目录。
3. 进入任意章节，运行对应 `.py` 文件。

示例：

```bash
cd 02_Python入门语法
python 01_Python中的字面量.py
```

## 部分章节依赖

部分代码可能依赖以下第三方库：

- `pyecharts`（可视化）
- `pymysql`（MySQL 操作）
- `pyspark`（Spark 相关）

可按需安装：

```bash
pip install pyecharts pymysql pyspark
```
